# AnimaTeX Studio

**TeXmExDeX Type Tools** - Create animated GIFs from images with keyframe animation.

## Features

- Import images and create keyframe animations
- Smooth motion interpolation between frames
- Export as animated GIF
- Adjust FPS, opacity, rotation, flip, and more
- Layer management with z-index control
- Visual timeline editor
- Real-time preview

## Quick Start

### Install Dependencies
```bash
npm install
```

### Run Development Server
```bash
npm run dev
```

### Build for Production
```bash
npm run build
```
